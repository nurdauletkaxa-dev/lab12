 Приложение для имитации входа с использованием фейкового JWT

 Тема и цель работы
Тема работы — создание веб-приложения, которое реализует имитацию процесса входа пользователя с использованием JWT-токенов. Цель работы — изучить основы работы с токенами, понять, как хранить их в браузере с использованием `localStorage`, а также реализовать простой механизм аутентификации на клиентской стороне.

Краткое описание программы
Программа представляет собой простое веб-приложение, в котором реализованы следующие функции:
1. Форма входа с полями для ввода email и пароля.
2. При успешном входе генерируется фейковый JWT-токен, который сохраняется в `localStorage`.
3. При перезагрузке страницы проверяется наличие токена. Если токен найден, приложение автоматически "авторизует" пользователя.
4. Кнопка "Выход" удаляет токен и возвращает пользователя на форму входа.

Приложение имитирует процесс аутентификации, не взаимодействуя с сервером. Токен хранится на клиенте для демонстрации концепции.
